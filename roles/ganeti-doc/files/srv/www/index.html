Ganeti

Ganeti is a cluster virtual server management software tool built on top of existing virtualization technologies such as Xen or KVM and other open source software. Ganeti requires pre-installed virtualization software on your servers in order to function. Once installed, the tool assumes management of the virtual instances (Xen DomU). Ganeti controls

  • Disk creation management,
  • Operating system installation for instances (in co-operation with OS-specific install scripts), and
  • Startup, shutdown, and failover between physical systems.
Ganeti is designed to facilitate cluster management of virtual servers and to provide fast and simple recovery after physical failures using commodity hardware.

The Ganeti documentation is hosted at http://docs.ganeti.org and includes an Installation tutorial as well as all man pages. Release tar balls and other related information can be downloaded from http://downloads.ganeti.org. They are signed with the Ganeti Release Signing Key 6AA8276A.

Ganeti is open source. The public git repository and the repositories of related projects can be found at http://git.ganeti.org, or simply

git clone git://git.ganeti.org/ganeti.git
Development is done on the ganeti-devel Mailinglist. The Development Process and other relevant information is described in the Wiki. Bugs a tracked on our public issue tracker.
